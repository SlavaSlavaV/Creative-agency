Animation description

This is a simple demo of jqueryIntroLoader using "gifLoader" animation. 

$(document).on('ready', function() {
    $("#element").introLoader({
        animation: {
            name: 'gifLoader',
            options: {
                ease: "easeInOutCirc",
                style: 'light',
                delayBefore: 500,
                delayAfter: 0,
                exitTime: 300
            }
        }
    });
});
